# AgenticAI
This is a simple project containing two different sections:
1. A simple Telegram bot using python-telegram-bot
2. A Django REST Framework API for managing messages

Each task has its own folder and README with setup instructions.

---

### Clone to the Repo
```bash
git clone <repository-url>
```

### Create a virtual environment
```bash
python -m venv venv
source venv/bin/activate  # for Linux and Mac
venv\Scripts\activate     # Windows
```


### Install dependencies
```bash
pip install -r requirements.txt
```
